# Video Info 

Electron base video info desktop application take info like 
frame rate, size, codec and other

## Dependencies 
- [FFmpeg](https://ffmpeg.org/)
- [fluent-ffmpeg](https://www.npmjs.com/package/fluent-ffmpeg)

## How to install

- clone the repo
- cd into repo
```bash
cd videoinfo-electron
```
- run npm
```bash
npm start
```
- start app
```bash
npm run electrin
```
